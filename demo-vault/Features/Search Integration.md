# Search Integration

Selecting a term in the cloud should connect naturally to search workflows.

## Workflow

1. User selects a token in the cloud
2. Plugin applies a search action
3. User explores matching notes and context

## What to verify

- Search text is escaped and applied safely
- Repeated clicks update search as expected
- Search results reflect current filter settings
