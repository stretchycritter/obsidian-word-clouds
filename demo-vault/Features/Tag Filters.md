# Tag Filters

Tag filters keep noise out of the cloud by excluding files or sections that should not contribute.

## Typical exclusions

- `#template`
- `#archive`
- `#private`
- `#reference`

## Expected outcome

- Better semantic signal
- More accurate topic clouds
- Less distortion from utility notes

## Edge cases to watch

- Notes with many tags and mixed content
- Nested folders with inconsistent tagging
- Files that use tags only in frontmatter
