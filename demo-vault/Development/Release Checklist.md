# Release Checklist

## Before version bump

- Build plugin assets successfully
- Confirm views load in demo vault
- Run checks from [[Testing/Test Plan]]

## Documentation pass

- Update [[Project/Plugin Overview]] if behavior changed
- Update [[Project/Roadmap]] priorities
- Confirm [[00 - Word Clouds Plugin Home]] links still represent current structure

## Final validation

- Check `manifest.json` version and compatibility
- Verify styles render cleanly in light and dark themes
- Confirm no broken wiki links in demo vault
